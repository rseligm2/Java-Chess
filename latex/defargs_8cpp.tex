\subsection{Detailed Description}
This scanner is used to convert a string into a list of function or template arguments. Each parsed argument results in a \mbox{\hyperlink{struct_argument}{Argument}} struct, that is put into an \mbox{\hyperlink{class_argument_list}{Argument\+List}} in declaration order. Comment blocks for arguments can also be included in the string. The argument string does not contain new-\/lines (except inside any comment blocks). An \mbox{\hyperlink{struct_argument}{Argument}} consists of the string fields\+: type,name,default value, and documentation The \mbox{\hyperlink{struct_argument}{Argument}} list as a whole can be pure, constant or volatile.

Examples of input strings are\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{stringliteral}{"(int a,int b) const"}}
\DoxyCodeLine{\textcolor{stringliteral}{"(const char *s="}hello world\textcolor{stringliteral}{",int=5) = 0"}}
\DoxyCodeLine{\textcolor{stringliteral}{"<class T,class N>"}}
\DoxyCodeLine{\textcolor{stringliteral}{"(char c,const char)"}}
\end{DoxyCode}


Note\+: It is not always possible to distinguish between the name and type of an argument. In case of doubt the name is added to the type, and the match\+Argument\+List in util.\+cpp is be used to further determine the correct separation.
